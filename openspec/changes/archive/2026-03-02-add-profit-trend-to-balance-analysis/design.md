## Context

当前理财详情页趋势分析已支持余额与净值的图表/列表展示，但余额模块仍只关注余额值本身，缺少收益维度。对于用户判断资金波动来源（净流入、收益贡献）不够直观。当前页面已有 `valueTrends`，其中包含余额与收益数据，可直接复用。

## Goals / Non-Goals

**Goals:**
- 在余额趋势模块补充“收益趋势”展示能力。
- 收益趋势在折线图与列表视图都可查看。
- 收益趋势与余额趋势按同一时间轴对齐，便于对比。
- 不新增接口，复用现有详情数据结构。

**Non-Goals:**
- 不改造净值趋势功能。
- 不引入新的后端聚合字段。
- 不调整详情页其他 Tab 的交互。

## Decisions

- 在余额趋势图表中新增收益序列（多折线），区分颜色并标注图例。
  - 备选：单独新增一个收益图卡片。
  - 取舍：与余额同图更利于同周期对比。

- 列表视图增加“收益”列，保持与图表使用同一原始数据并统一格式化。
  - 备选：收益使用独立列表。
  - 取舍：同表展示降低认知切换成本。

- 保持金额类两位小数规则（收益和余额一致），避免与净值四位精度规则混淆。
  - 备选：收益提升到四位小数。
  - 取舍：收益本质是金额，维持财务展示习惯。

## Risks / Trade-offs

- [Risk] 同图展示两条曲线可能造成视觉拥挤 → Mitigation：使用明确图例与颜色、支持 hover 高亮。
- [Risk] 收益可能出现负值导致读数理解偏差 → Mitigation：列表中保留正负号并在 tooltip 明确字段名。
- [Risk] 历史数据缺失收益字段时渲染异常 → Mitigation：收益字段做空值兜底为 0。
